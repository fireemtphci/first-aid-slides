# 🛠️ 萬用指令模板

> 適用於所有急救主題，修改「主題名稱」即可使用

---

## 📋 基本版（最常用）

```
請幫我製作一份急救教學簡報，主題是：【在這裡填入主題】

設定條件：
- 對象：國高中學生
- 風格：活潑易懂，多用 emoji 和圖示說明步驟
- 結構：封面、學習目標、核心步驟（每步驟一張投影片）、
        注意事項、總結複習、Q&A
- 每張投影片請給我：標題 + 重點內容 + 講師備註
- 輸出格式：PPTX 和 PDF 兩種
```

---

## 📝 只要大綱版

```
請給我「【主題名稱】」急救簡報大綱，
8～10張投影片，條列重點就好，不需要完整內容
```

---

## ✏️ 修改投影片版

```
第【N】張投影片請改成更簡單易懂的版本，
用步驟1／2／3的方式呈現，並加上 emoji 圖示
```

---

## 🎤 加入講師備註版

```
請幫每張投影片加上講師備註，
說明這張投影片該怎麼講解，
每張約30秒～1分鐘的內容
```

---

## 📝 出題測驗版

```
根據這份【主題名稱】簡報內容，
幫我出5題是非題＋3題選擇題，
並附上正確答案與解說
```

---

## 🔄 重新設計版

```
以下是我的急救簡報內容，
請保留所有資訊，但改成更活潑的視覺設計，
多用顏色區塊、emoji、步驟編號：

【貼上現有簡報內容】
```
