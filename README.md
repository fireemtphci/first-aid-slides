# 🚑 急救教學簡報指令模板庫

> 專為校園急救教學設計的 Claude 提示詞資料庫
> 使用方式：複製提示詞 → 貼到 Claude → 下載簡報

---

## 📁 資料夾說明

| 資料夾 | 內容 |
|--------|------|
| `prompts/` | 各主題的完整提示詞（直接複製使用） |
| `templates/` | 萬用指令模板（可自行修改） |
| `records/` | 已完成簡報紀錄 |

---

## 🚀 使用步驟

1. 進入 `prompts/` 資料夾
2. 點開想要的主題（例如：`CPR心肺復甦術.md`）
3. 複製全部內容
4. 貼到 [Claude](https://claude.ai) 對話框
5. 等待生成 → 下載 PPTX 和 PDF

## ✏️ 新增主題方式

1. 進入 `prompts/` 資料夾
2. 按 `Add file → Create new file`
3. 檔名格式：`主題名稱.md`
4. 複製任一現有主題的格式，修改內容即可
